# PWA Icon Generation Guide

## Quick Option: Use Online Generator

1. **Go to**: https://www.pwabuilder.com/imageGenerator
2. **Upload** a square logo/icon (at least 512x512px)
3. **Download** the generated icon pack
4. **Copy** all icons to `c:\EasyRetailer\client\public\icons\`

## Alternative: Use Favicon.io

1. **Go to**: https://favicon.io/favicon-converter/
2. **Upload** your logo
3. **Download** and extract
4. **Rename** files to match:
   - `android-chrome-192x192.png` → `icon-192x192.png`
   - `android-chrome-512x512.png` → `icon-512x512.png`
5. **Copy** to `public/icons/`

## Manual Creation (Photoshop/GIMP)

Create square PNG images with transparent background:
- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`
